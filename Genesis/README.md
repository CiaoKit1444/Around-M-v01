# Peppr Around — Genesis

> *"This folder is the soul of the platform. It contains everything we learned, everything we built, and everything we would do differently. Read it before you write a single line of code."*

---

## What Is This?

The Genesis folder is the living documentation of the Peppr Around platform — a hospitality service request management system built from scratch over 78 phases between January and March 2026. It was created as a Hall of Fame retrospective: a complete record of the platform's DNA so that any future incarnation starts with wisdom rather than from zero.

This is not a technical specification. It is a **memory** — the accumulated knowledge of every decision made, every mistake corrected, and every principle discovered during the building of this platform.

## How to Use This Folder

**If you are starting a new version of this platform:**
Read the documents in order. Start with the Manifesto to understand the purpose. Read the Architecture to understand the structure. Read the Lessons Learned to avoid the mistakes. Read the Phase Chronicle to understand the evolution. Read the Design Language to understand the aesthetics. Then follow the Rebirth Playbook sprint by sprint.

**If you are joining the existing platform:**
Read the Manifesto and the Lessons Learned first. They will give you the context to understand why the code looks the way it does and what constraints you must respect.

**If you are evaluating this platform as a product:**
Read the Manifesto and the Phase Chronicle. They tell the story of what was built and why.
